NUM-CODE QUICK GUIDE

Num-Code programs are stored in Num-Code.txt.
Every instruction is 8 numbers:

OPCODE A B C IMM0 IMM1 IMM2 IMM3

Variables: 0=A, 1=B, 2=C, ... 25=Z

0 Stop
1 Set variable
2 Add number
3 Subtract number
4 Multiply by number
5 Divide by number
6 Power
7 Print variable
8 Print number
9 Print text
10 Jump
11 Jump if zero
12 Jump if not zero
13 Jump if less
14 Jump if greater
15 Jump if equal
16 Add/compare/loop
17 Subtract/loop
18 Copy variable
19 Add variable
20 Import module
21 Call function with 0 inputs
22 Call function with 1 input
23 Call function with 2 inputs

Built-in modules:
1 Math, 2 Random, 3 Time, 4 Input, 255 all built-ins

External modules use IDs 128-254.
Module 128 filenames:
Windows: modules/Num-Code-Module-128.dll
Linux/Raspberry Pi: modules/Num-Code-Module-128.so
macOS: modules/Num-Code-Module-128.dylib

The included module 128 has:
1 double one number
2 add two numbers
3 return 42

The included Num-Code.txt prints:
42
31
42

BUILD
Windows x64: Build-Num-Code-Windows-x64.bat
Linux x64: ./Build-Num-Code-Linux-x64.sh
Intel macOS: ./Build-Num-Code-macOS-Intel.sh
Apple Silicon macOS: ./Build-Num-Code-macOS-Apple-Silicon.sh
Raspberry Pi 64-bit: ./Build-Num-Code-Raspberry-Pi-ARM64.sh

On Linux/macOS/Raspberry Pi, first run:
chmod +x Build-Num-Code-*.sh

Required tools:
Windows: Python, NASM, MinGW-w64/GCC
Linux: Python 3, NASM, GCC
Intel macOS: Python 3, NASM, Xcode command-line tools
Apple Silicon macOS: Python 3, Xcode command-line tools
Raspberry Pi: Python 3, GCC

Only load external libraries you trust.
